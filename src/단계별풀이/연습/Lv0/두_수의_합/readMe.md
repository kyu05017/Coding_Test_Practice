# 프로그래머스 Lv.0 -  두 수의 합

--------------------

### 문제 내용

    정수 [num1]과 [num2]가 주어질 때, 
    [num1]과 [num2]의 합을 return하도록 soltuion 함수를 완성해주세요.
*****
### 제한사항

    1. -50,000 ≤ [num1] ≤ 50,000

    2. -50,000 ≤ [num2] ≤ 50,000


*****
### 입출력 예시

|    | num1 | num2 | result |
|----|------|------|--------|
| #1 | 2    | 3    | 5      |
| #2 | 100  | 2    | 102    |

*****

### 입출력 예 설명
###### 입출력 예 #1
    1. [num1]이   2이고 [num2]가 3이므로   2 + 3 =   5를 return합니다.

    2. [num1]이 100이고 [num2]가 2이므로 100 + 2 = 102를 return합니다.
